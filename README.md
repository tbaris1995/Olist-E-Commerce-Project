# Olist E-Commerce Data Analysis

## Project Overview
This project analyzes a real-world e-commerce dataset to extract business insights and support data-driven decision-making.

## Tools & Technologies
- SQL
- Data Analysis
- Data Visualization

## Data Cleaning
- Handled missing and inconsistent values
- Removed duplicate records
- Standardized text fields (city, state)
- Normalized date formats

## Key KPIs
- Average Order Value (AOV)
- Customer Lifetime Value (CLV)
- Average Order Interval
- Regional Revenue Distribution
- Delivery Time Analysis
- Top Selling Product Categories

## Key Insights
- A small group of loyal customers generates a significant portion of revenue
- Order volume increases during year-end periods (seasonality effect)
- Customer spending is highly skewed
- São Paulo (SP) is the top revenue-generating region

## Business Recommendations
- Focus on high-value customers with loyalty programs
- Improve delivery times in slower regions
- Increase promotions in high-performing categories
- Monitor negative reviews and improve customer experience
